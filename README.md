# Aura Launcher

Aura Launcher is a desktop Minecraft launcher and instance manager built with Tauri, Rust, React and TypeScript.

It is designed to manage Minecraft instances and browse/install content from Modrinth and CurseForge, including mods, modpacks, resource packs and shaders.

## Development

```powershell
npm install
npm run dev
```

## Desktop Build

```powershell
npm run tauri:build
```

CurseForge production integration should use an approved Aura-owned API key or backend infrastructure. End users should not need to enter their own API key.
